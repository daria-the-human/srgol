#Услуги для физических лиц

Файл блока:

```
layout/main-page/services/services.twig
```

Здесь находятся карточки услуг.

Каждая услуга:


- содержит ссылку
- ведет на отдельную страницу услуги (статью)

---

### Где хранятся страницы услуг

```
layout/articles/
```

Каждый файл = отдельная услуга (статья).

---

### Как добавить новую услугу

Блок услуг на главной странице управляется **массивом `services`**.  
Менять нужно только данные в массиве — верстку не трогать.

Пример:

```twig
{% set services = [
  {
    title: 'Услуги семейного юриста',
    icon: 'divorce',
    list: [
      { text: 'оформление развода', href: '/error.html' }
    ]
  }
] %}
```

---

### Добавить новую карточку услуг

1. Скопировать любой объект `{ ... }` в массиве `services`
2. Поменять:
    - `title`
    - `icon`
    - `list`

⚠ Если карточек станет больше — верстка может поехать.  
Проверить главную после изменений.

---

### Добавить услугу в карточку

Внутри `list` добавить элемент:

```twig
{ text: 'Новая услуга', href: '/error.html' }
```

---

### Ссылки на услуги

Если страница услуги есть → ссылка на неё:

```twig
href: '/articles/divorce.html'
```

Если страницы нет → заглушка:

```twig
href: '/error.html'
```

---

### Где лежат страницы услуг

Twig-шаблоны:

```
source/layout/articles/
```

После сборки:

```
source/articles/
```

Редактировать нужно Twig, не HTML.

---

### Иконки

`icon: 'car'` → используется `car.svg`  
Имя должно совпадать с файлом иконки.

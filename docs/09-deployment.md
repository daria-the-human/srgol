# Публикация сайта (Deploy)

Сайт публикуется вручную через загрузку собранной версии на сервер.

---

## 1. Собрать сайт

В корне проекта выполнить:

```bash
gulp build
```

Появится папка:

```
build/
```

---

## 2. Загрузить на сервер

Лучше удалить старые файлы и загрузить **содержимое** папки:

```
build/
```

Это поможет избежать багов из-за кэша.

---

## 3. Проверить сайт

После загрузки проверить:

- главную страницу
- услуги
- блог
- контакты
- кнопки и ссылки
- изображения

# Управление контентом сайта

В этом разделе описано, где и как редактируется содержимое сайта:

- тексты
- услуги
- статьи
- блог
- ссылки
- блоки главной страницы

Все изменения выполняются в Twig-файлах проекта.  
HTML редактировать не нужно.

## Типы контента

### Главная страница 

[Документация по главной странице](06-main-page.md)

### Статьи услуг
- связаны с услугами
- ведут с главной
- лежат в `layout/articles`

[Документация по услугам на главной странице](07-services.md)

### Статьи блога
- отдельные материалы
- не привязаны к услугам

[Документация по блогу](08-blog.md)

### Кейсы

Раздел «Кейсы» пока не реализован.

Возможные варианты:

- добавить страницу кейсов
- оформить кейсы как блог-статьи
- удалить пункт из меню

---

## Важно

После любых изменений:

```bash
gulp
```

Чтобы опубликовать сайт:

```bash
gulp build
```

Загрузить на сервер папку:

```
build/
```

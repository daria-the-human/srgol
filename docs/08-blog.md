# Блог

Блог — это раздел для статей,  
которые **не относятся напрямую к услугам**.

# Блог

Блог состоит из:

- страницы списка статей (карточки)
- отдельных страниц статей

---

## Страница списка блога (blog-page)

Файл:

```
source/layout/blog-page/blog-page.twig
```

На странице выводятся карточки статей:
- заголовок
- описание
- изображение
- ссылка на статью

Карточки управляются массивом `blogList`.

Пример:

```twig
{% set blogList = [
  {
    title: 'Название статьи',
    text: 'Краткое описание',
    href: 'blog/article-slug/',
    image: './assets/images/blog-1.jpg'
  }
] %}
```

---

### Добавить карточку статьи

В `blog-page.twig` добавить новый элемент в массив `blogList`:

```twig
{
  title: 'Новая статья',
  text: 'Краткое описание',
  href: 'blog/new-article/',
  image: './assets/images/blog-new.jpg'
}
```

---

## Страница статьи блога

Статьи лежат в папке:

```
source/layout/blog/
```

Каждая статья — отдельная папка.

Пример:

```
layout/blog/court-order/
```

Внутри находится файл:

```
article.twig
```

---

### Добавить новую статью

1. Скопировать папку любой существующей статьи
2. Переименовать папку (это будет URL)
3. Изменить контент в `article.twig`
4. Добавить карточку в `blog-page.twig`
5. Запустить `gulp`

---

## Связь карточки и статьи

Поле `href` в `blogList` должно совпадать с папкой статьи.

Пример:

```
href: 'blog/court-order/'
```

→ папка:

```
layout/blog/court-order/
```

# Разница: услуги vs блог

**Статьи услуг**
- связаны с услугами
- ведут с главной
- лежат в `layout/articles`

**Статьи блога**
- отдельный контент
- не привязан к услугам
- лежит в `layout/blog`

# Структура проекта

В проекте используется разделение на исходный код (разработка) и сгенерированные файлы (готовый сайт).

Основная разработка ведётся в папке `source/`.  
Итоговая версия сайта формируется в папке `build/`.

---

## Общая структура

```
project-root
├── gulpfile.js
├── package.json
├── package-lock.json
├── .gitignore
├── README.md
├── source/
└── build/
```

---

## Корневые файлы

### gulpfile.js
Конфигурация системы сборки Gulp.  
Определяет задачи компиляции Twig, SCSS, сборки скриптов и production-сборки.

### package.json
Описание зависимостей проекта и npm-скриптов.

### package-lock.json
Зафиксированные версии зависимостей.

### .gitignore
Список файлов и папок, исключённых из Git.

### README.md
Основной файл документации и навигации.

---

## Папка source/

Основная папка разработки.  
Все изменения вносятся именно здесь.

```
source/
├── layout/
├── scss/
├── js/
├── assets/
├── articles/
├── blog/
├── fonts/
├── images/
└── *.html
```

---

## layout/

Twig-шаблоны сайта.

⚠ HTML в `source/*.html` не редактируется напрямую — он генерируется из Twig.

```
layout/
├── main-page/
├── blog/
├── articles/
├── blog-page/
├── components/
├── error-page/
```

Назначение:

- **main-page/** — шаблон главной страницы
- **blog/** — шаблон страницы блога (статья)
- **blog-page/** — шаблоны страницы блога (список статей)
- **articles/** — шаблоны страниц услуг (статья по теме услуги)
- **components/** — переиспользуемые блоки
- **error-page/** — страница 404

---

## scss/

Исходные стили проекта.

```
scss/
├── style.scss
├── colors.scss
├── fonts.scss
├── grid.scss
├── libs.scss
├── media.scss
├── extra.scss
└── sections/
```

Назначение:

- **style.scss** — главный файл, импортирует остальные
- **colors.scss** — переменные цветов
- **fonts.scss** — подключение шрифтов
- **grid.scss** — сетка
- **libs.scss** — сторонние стили
- **media.scss** — миксины адаптива
- **extra.scss** — базовые глобальные стили
- **sections/** — стили отдельных блоков

После сборки стили попадают в:

```
assets/css/style.min.css
```

---

## js/

JavaScript проекта.

```
js/
├── main.js
├── libs.min.js
└── min/
```

Назначение:

- **main.js** — основной скрипт сайта
- **libs.min.js** — сторонние библиотеки
- **min/** — минифицированные версии

---

## assets/

Готовые ресурсы сайта.

```
assets/
├── css/
├── fonts/
└── images/
```

Назначение:

- **css/** — скомпилированные стили
- **fonts/** — шрифты
- **images/** — изображения

---

## images/icons/

SVG-иконки проекта.

Используются для формирования SVG при сборке.

---

## articles/

HTML-страницы услуг (сгенерированные из Twig).

Связаны с блоком «Услуги» на главной странице.

---

## blog/

HTML-страницы блога (сгенерированные из Twig).

Не связаны с услугами, используются только в разделе блога.

---

## fonts/

Исходные файлы шрифтов (woff/woff2).

---

## images/

Исходные изображения проекта.

---

## build/

Production-версия сайта.

Формируется командой:

```bash
gulp build
```

На сервер загружается только содержимое этой папки.
